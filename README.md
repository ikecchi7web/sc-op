# sc-op

SwipeCardで作成したHTMLプロトタイプの公開用リポジトリです。

## デモ

- `demos/swipecard-live/`
  - 縦スワイプと▲▼、タップ式の横展開、Live操作、ページマップ、簡易CMSを備えたPWA試作
  - 縦行×横列によるページ枠の自動追加・削除に対応
- `demos/presen02-recompose/`
  - 分離画像をHTML/CSSで再配置した再構成テスト

ルートの `index.html` から各デモを開けます。GitHub Pagesにもそのまま配置できる静的構成です。

## 公開用の調整

- APIキー、認証情報、個人情報を含めていません
- `sc_pre` の内部資料、ローカル設定、Git履歴は含めていません
- 実在人物を含む背景画像は公開版から除外しました
- SwipeCard LiveのCMSから、権利を確認した画像をローカルで指定できます
- QR生成にはMITライセンスの`qrcode-generator 1.4.4`を同梱しています
- デモ最終ページには、本人の許可を得た公開用vCardとプロフィール画像を含みます

## ローカルで開く

```sh
python3 -m http.server 4174
```

その後、`http://127.0.0.1:4174/` を開きます。
